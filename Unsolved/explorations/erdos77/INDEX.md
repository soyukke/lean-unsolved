# エルデシュ問題 #77: Ramsey Number Limit（ラムゼー数の極限）探索インデックス

## 問題概要
lim_{k→∞} R(k)^{1/k} を求めよ。R(k)は対角ラムゼー数。

- 既知の範囲: √2 ≤ liminf ≤ limsup ≤ 3.7992
- Erdős-Szekeres (1935): R(k) ≤ C(2k-2, k-1) → 上界極限 4
- Erdős (1947): R(k) ≥ 2^{k/2} → 下界極限 √2
- Campos-Griffiths-Morris-Sahasrabudhe (2023): R(k) ≤ (4-ε)^k (ブレークスルー)
- Erdős予想: 極限は2かもしれない
- 懸賞: $100（極限の存在）、$250（値の決定）

## 主要な知見

1. **R(k)^{1/k} は緩やかに増加**: k=3で1.82, k=4で2.06, k=5で[2.12, 2.17]。kが大きくなると上界と下界の開きが増大
2. **回避グラフの自己相補性**: K_5での12通りの回避塗り分けは全て次数列(2,2,2,2,2)でC_5型。Paley(17)も自己相補的で8-正則
3. **Paley構成のクリーク数**: Paley(17)のクリーク数=独立数=3で、単色K_4を回避 → R(4) ≥ 18
4. **2023年のブレークスルー**: 88年間動かなかった上界4を初めて改善。現在の最良: ≤ 3.7992^k
5. **ギャップ**: 下界√2と上界3.7992の間は依然広い。Erdős予想の2は現在の下界√2よりわずかに上
6. **Paley graph系列**: G(5)〜G(61)を構成しω,αを計算。G(5):ω=2→R(3)≥6(tight)、G(17):ω=3→R(4)≥18(tight)、G(41):ω=5→R(6)≥42。自己相補性が根本的性質
7. **Paley構成の限界**: ω(G(q))≈(1/2)log₂(q)で対数的成長。確率的下界R(k)≥2^{k/2}と同オーダーで、R(k)^{1/k}→√2しか与えない
8. **下界改善の停滞**: Erdős(1947)以来77年間、下界の基底√2は改善されていない。上界は2023年に88年ぶりに4から改善
9. **R(k)^{1/k}下界のピーク**: k=5付近で約2.12のピークを持ち、その後k=10で1.95まで微減。上界は緩やかに増加(k=10で2.74)
10. **三角形数公式**: T(q)=q(q-1)(q-5)/48がq=5〜61の全素数で一致確認。三角形密度は1/8≈0.125に漸近
