# 探索106: Baker定理の量的形式によるサイクル排除の系統的拡張

## メタデータ
- **カテゴリ**: 論理的/モデル理論
- **成果レベル**: 中発見
- **日付**: 2026-03-27
- **スクリプト**: `scripts/collatz_baker_cycle_exclusion.py`

## 目的
探索088で周期2のサイクルが |n_0| < 0.64 < 1 で排除された。Baker定理 |2^s - 3^p| > exp(-c·max(s,p)) を用いて、各周期pに対する最適なsを計算し、排除可能な周期の範囲を系統的に拡張する。先行研究(Steiner, Simons-de Weger, Hercher)との比較も行う。

## アプローチ
1. **小周期の完全パターン列挙 (p=1..10)**: 全 (v_1,...,v_p) パターンで n = C/D を計算し、整数解の整合性を検証
2. **保守的排除 (p=1..200)**: 3^p / |2^s - 3^p| < 2 で排除判定
3. **Baker下界 (LMN)**: Laurent-Mignotte-Nesterenko (1995) の理論下界と実際の |2^s-3^p| を比較
4. **連分数展開**: log_2(3) の連分数近似から排除困難な周期を特定

**核心的事実**: 合計3,319,527パターンを検査し、C/D が整数かつ整合的なサイクルは1つも存在しない。max(C/D) は非整数値のため、排除が成立。

### 2. 保守的排除 (p=1..200)

条件: 3^p / |2^s - 3^p| < 2 (LLLなし)

- 排除成功: **83/200 周期 (41.5%)**
- 排除できない周期: 117個

排除が成功する周期: gap = log_2(3^p) - log_2(|2^s-3^p|) < 1 となるもの。
例: p=2 (gap=0.363), p=4 (0.785), p=7 (0.196), p=12 (0.039), p=24 (0.079)

### 3. Baker下界 vs 実際の値

| p | s | 実際 log2|D| | Baker下界(LMN) | 余裕(ビット) |
|---|---|-------------|---------------|------------|
| 2 | 4 | 2.81 | -11789 | 11792 |
| 7 | 12 | 10.90 | -11781 | 11792 |
| 53 | 85 | 84.00 | -11708 | 11792 |
| 200 | 317 | 309.41 | -11475 | 11785 |

**Baker理論下界は実際の値より約11,800ビット小さい** — LLL格子簡約なしでは全く排除力がない。

### 4. 連分数展開と困難な周期

log_2(3) = [1, 1, 1, 2, 2, 3, 1, 5, 2, 23, 2, 2, 1, 1, 55, ...]

収束子 s/p が |2^s - 3^p| を最小にする:
- p=5 (s=8): gap = 4.22
- p=12 (s=19): gap = 6.22
- p=41 (s=65): gap = 6.44
- p=53 (s=84): gap = 8.91
- p=306 (s=485): gap = 9.93

これらの周期が排除で最も困難。

## 新しい視点と発見

1. **パターン列挙の威力**: p ≤ 10 で合計330万パターンを全検査し、整数解の整合性検証で完全排除。整数解不整合が0件であることは、C/D が整数になることすら稀であることを示す。

2. **Baker理論の限界**: LMN下界は実効値から11,800ビット離れており、「素」のBaker定理では排除力がない。LLL格子簡約が本質的。

3. **保守的排除の41.5%**: LLLなしでも |2^s - 3^p| が 3^p/2 を超える周期が約4割存在。これらは |2^s - 3^p| が相対的に大きい周期（gap < 1）。

4. **困難な周期の構造**: log_2(3) の連分数近似の分母（p=1, 2, 5, 12, 41, 53, 306, ...）で |2^s-3^p| が相対的に最小になり、排除が最も困難。

5. **Z_2上の豊富な構造との対比**: 探索088で確認した「分母2^s-3^pの奇数性→Z_2上の形式的周期点が無限」との整合性: 正整数への篩い分けが本質的困難。

## 結論

- 小周期(p≤10)は完全パターン列挙+整合性検証で厳密に排除
- Baker理論下界のみでは排除力が極めて弱い（11,800ビットの余裕）
- LLL格子簡約(Simons-de Weger)で p≤68、計算的方法(Hercher)で p<10^{11}
- 全周期の理論的排除にはBakerを超える新手法が必要
- log_2(3)の有理近似の質が排除の本質的障壁

## 形式化への道筋
- p≤10の完全排除は原理的にLean形式化可能（有限パターンの全検査）
- v2(2^s-3^p)=0（分母の奇数性）は既に形式化の道筋あり（探索088）
- Baker下界の形式化は困難（超越数論的結果）
