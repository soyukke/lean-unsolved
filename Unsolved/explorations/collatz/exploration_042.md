# 探索42: 大規模 d/u 最小値の精密計算

**日付**: 2026-03-18
**スクリプト**: `scripts/collatz_du_large.py`
**計算範囲**: n = 3..10^7 の奇数（4,999,999個、108秒）

## 実験設計

コラッツ予想において、奇数 n の軌道の偶数ステップ数 d と奇数ステップ数 u の比 d/u が
log₂(3) ≈ 1.58496 を超えるかどうかを大規模に検証する。

d/u > log₂(3) が全ての n で成立すれば、2^d > 3^u → n は必ず減少 → 収束が保証される。

### 調査項目
1. n ≤ 10^7 の全奇数で最終 d/u の最小値追跡（上位100個）
2. メルセンヌ数 2^k - 1 (k=2..30) の特別調査
3. 10^k ごとの最小 d/u の推移
4. 最小 d/u を持つ n の2進表現の特徴
5. 途中の累積 d/u が log₂(3) を下回る瞬間の統計

## 主要結果

### 1. 最終 d/u < log₂(3) となる n: **0個**

n ≤ 10^7 の全4,999,999個の奇数で、最終的な d/u は全て log₂(3) を超える。

### 2. 全体の最小 d/u

| 指標 | 値 |
|------|-----|
| 最小 d/u | 1.6757812500 |
| 達成する n | 8,400,511 |
| d | 429 |
| u | 256 |
| d/u - log₂(3) | 0.0908 |
| n の2進表現 | 100000000010111001111111 |
| n の1の密度 | 0.500 |

### 3. 10^k ごとの最小 d/u の推移

| 範囲 | 最小 d/u | n* | d/u - log₂(3) | 縮小率 |
|------|----------|-----|---------------|--------|
| n ≤ 10^1 | 2.1667 | 9 | 0.5817 | - |
| n ≤ 10^2 | 1.7073 | 27 | 0.1224 | 4.75x |
| n ≤ 10^3 | 1.7073 | 27 | 0.1224 | - |
| n ≤ 10^4 | 1.7073 | 27 | 0.1224 | - |
| n ≤ 10^5 | 1.7073 | 27 | 0.1224 | - |
| n ≤ 10^6 | 1.6872 | 837,799 | 0.1022 | 1.20x |
| n ≤ 10^7 | 1.6758 | 8,400,511 | 0.0908 | 1.13x |

**観察**: 差分は緩やかに縮小しているが、10^5 → 10^6 → 10^7 の縮小率は
1.20x → 1.13x と減速傾向にある。0に収束する兆候は見られない。

### 4. メルセンヌ数 2^k - 1 の d/u

メルセンヌ数は全て d/u > log₂(3) を満たす。最小差分の傾向:

| k | d/u - log₂(3) | 備考 |
|---|---------------|------|
| 23 (8388607) | 0.1334 | 調査範囲で最小 |
| 25 (33554431) | 0.1558 | |
| 29 (536870911) | 0.1805 | |

メルセンヌ数の d/u は k に対して 1.7〜1.8 付近で振動し、log₂(3) から十分離れている。

### 5. 途中の累積 d/u < log₂(3) の統計

**約42.6%** の n で、軌道の途中で累積 d/u が一時的に log₂(3) を下回る。

| 範囲 | 途中で下回る割合 |
|------|-----------------|
| n ≤ 100 | 40.82% |
| n ≤ 1,000 | 54.71% |
| n ≤ 10,000 | 43.01% |
| n ≤ 100,000 | 42.72% |
| n ≤ 1,000,000 | 42.56% |
| n ≤ 10,000,000 | 42.64% |

途中の最小 d/u は 1.067 (n=32767=2^15-1) まで下がるが、最終的には必ず回復する。
これは「復元力」（探索30で発見）の存在を裏付ける。

### 6. 最小 d/u を持つ n の2進表現の特徴

| 指標 | d/u最小上位100 | ランダム |
|------|--------------|---------|
| 1の密度（平均） | 0.5596 | 0.5318 |

- d/u が小さい n は、やや1の密度が高い傾向があるが、大きな偏りはない
- 特定のビットパターン（例: 末尾に連続1）は見られるが、決定的ではない
- n=8400511 の末尾7ビットは1111111（連続1）→ 連続上昇フェーズが長い

## 分析

### 核心的発見

1. **d/u > log₂(3) は n ≤ 10^7 で例外なし**: 最小差分 0.0908 は log₂(3) から約5.7%上
2. **差分の縮小は減速**: 10^k が増えるにつれ差分は縮小するが、縮小率自体が減少
3. **途中では42%が下回る**: 最終値では全て超えるが、途中では約4割が一時的に下回る。これは「復元力」のおかげで最終的に回復する
4. **メルセンヌ数は安全**: 2^k-1 形は d/u が比較的高く、最小差分 0.13 程度

### d/u の下界に関する仮説

差分の推移 (0.58 → 0.12 → 0.10 → 0.09) を見ると:
- 差分は O(1/log(N)) 程度で縮小する可能性がある
- しかし、0に収束するかは不明
- もし差分が正の定数 c > 0 に収束するなら、d/u > log₂(3) は全 n で成立

### d/u ≈ 429/256 の構造

最小値 n=8400511 の d/u = 429/256 は、u=256=2^8 が2の冪。
これは256回の奇数ステップを経る長い軌道で、偶数ステップの「余剰」が最小限になるケース。
