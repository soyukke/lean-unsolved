# 探索049: 逆Syracuse木の各深さでの密度と成長率の精密解析

## 日付
2026-03-18

## 目的
逆Syracuse木（逆Collatz木）を根 n=1 から展開し、各深さでのノード数・成長率・被覆密度を精密に測定する。コラッツ予想は「逆木が全正整数を含む」ことと同値であり、その被覆の進行パターンを定量化する。

## 手法
- **逆Syracuse写像**: T(n)=n/2 (偶数), (3n+1)/2 (奇数) の逆をBFSで展開（深さ35まで）
- **逆Collatz写像**: C(n)=n/2 (偶数), 3n+1 (奇数) の逆をBFSで展開（深さ50まで）
- 各深さでノード数、成長率、被覆率、新規ノードの特徴を測定

## 結果

### 1. 成長率の収束

| 写像 | 平均成長率 | 後半10ステップ平均 | 理論値 |
|------|-----------|-------------------|--------|
| Syracuse逆 | 1.3273 | 1.3344 | 4/3 ≈ 1.3333 |
| Collatz逆 | 1.2645 | 1.2638 | (理論的に ≈ 4/3^{1/k}) |

- **Syracuse逆木**: 成長率は4/3に非常によく収束。深さ25以降は1.33±0.01の範囲で安定。
- **Collatz逆木**: 成長率は約1.264に収束。Syracuse木の成長率の「半分ステップ」（偶数→偶数の2倍化ステップを含むため）。

### 2. 被覆率の推移

| 範囲 M | 深さ50時点の被覆率 (Collatz木) |
|--------|-------------------------------|
| [1, 1,000] | 56.9% |
| [1, 10,000] | 30.6% |
| [1, 100,000] | 13.1% |
| [1, 1,000,000] | 3.9% |

**重要な観察**: 被覆率は M が大きくなるほど低下する。これは逆木の最大値が指数的に成長（2^d）するのに対し、ノード数は (4/3)^d 程度でしか増えないため。深さ d での被覆率は概ね O((4/3)^d / 2^d) = O((2/3)^d) → 0 に向かう。

しかし、これは特定の深さ時点での「スナップショット」であり、d→∞ では全正整数が含まれることが予想の主張。

### 3. 理論的成長率との比較

Syracuse逆木で、累積ノード数 / (4/3)^d の比率は約 **2.69** に収束する傾向。

```
累積ノード数 ≈ 2.69 × (4/3)^d
```

この定数 ≈ 2.69 は逆木の「分岐効率」を反映している。

### 4. 新規ノードの特徴 (Collatz木)

| 特徴 | 安定値 (深さ30以上) |
|------|-------------------|
| 奇数の割合 | ≈ 21% |
| 偶数の割合 | ≈ 79% |
| 3で割れる割合 | ≈ 33% |

- 偶数が圧倒的に多い（逆写像 n→2n が常に適用可能なため）
- 3の倍数の割合は自然密度1/3と一致（偏りなし）

### 5. 遅延到達整数の分析

[1, 10,000] 内で深さ50まででの到達率はわずか **30.6%**。

深さ30以上で初到達する小さな整数の例:
- 39 (深さ34), 57 (深さ32), 123 (深さ46), 127 (深さ46)

**遅延到達の特徴**:
- mod パターンに顕著な偏りはない（各剰余類がほぼ均等）
- ビット密度（1の割合）は0.4〜0.6に集中（極端な密度の数は早く到達する傾向）
- 2進表現で上位ビットの構造よりも、コラッツ軌道の長さが到達深さを決定

### 6. 被覆密度の漸近的振る舞い

[1, 100] 範囲は深さ30で84%に到達し、その後停滞。これは残りの16%がさらに深い探索を必要とすることを示す。

各範囲 M での被覆率は深さに対してほぼ**線形に増加**:
- [1, 1,000]: 深さ50で56.9%（1深さあたり約+1.1%）
- [1, 10,000]: 深さ50で30.6%（1深さあたり約+0.6%）
- [1, 100,000]: 深さ50で13.1%（1深さあたり約+0.26%）

## 考察

### 成長率 4/3 の意味
- 各ステップで、「2倍」の分岐は確実に起こり、「(2n-1)/3」の分岐は約1/3の確率で起こる
- 期待される成長率 = 1 + 1/3 = 4/3（Syracuse写像の場合）
- この値は探索17の結果と完全に整合

### 被覆の「遅さ」の本質
- ノード数は (4/3)^d で増加するが、最大値は 2^d で増加
- したがって密度は (2/3)^d で減少 → 各深さでは「疎」
- しかし、Σ_{d=0}^{∞} (4/3)^d = ∞ なので、十分な深さで全整数をカバー可能
- これが予想の「もっともらしさ」の確率的根拠

### 逆木の「フラクタル的」構造
- [1, M] の被覆率が M に依存することは、逆木がスケール不変ではないことを示す
- 小さい M ほど早く被覆される（小さい数は短い軌道で1に到達）
- 大きい M の被覆には指数的に深い探索が必要

## Lean形式化への示唆
- `InverseCollatzTree (d : Nat) : Finset Nat` — 深さ d までの逆木ノード集合
- `growth_rate_bound`: N(d+1)/N(d) の上下界を証明
- `coverage_monotone`: 被覆率が深さに関して単調増加であることの証明（自明だが基盤として有用）
