# 探索053: Syracuse軌道の形式的冪級数の有理性テスト

## 日付
2026-03-18

## 目的
Syracuse軌道 a_0, a_1, a_2, ... の生成関数 F_n(x) = Σ a_k x^k が有理関数であるかをテストする。もし F(x) が有理関数なら軌道は最終的に周期的（コラッツの帰結）。Pade近似、線形再帰検出（Berlekamp-Massey的）、Z変換の極解析、Dirichlet級数の収束性を用いて多角的に検証する。

## 手法
1. **Pade近似 [m/n]**: F(x) を P_m(x)/Q_n(x) で近似し、残差の減衰を評価
2. **線形再帰検出**: 軌道列が a_k = c_1 a_{k-1} + ... + c_d a_{k-d} を満たすか最小二乗法で検出
3. **Z変換の極解析**: Prony法で F(z) = Σ a_k z^{-k} の極を推定
4. **Dirichlet級数**: D_n(s) = Σ T^k(n) / k^s の収束横軸を推定
5. **対数軌道解析**: log(a_k) のトレンドと残差の周期性を検査

テスト値: n = 7, 27, 97, 171, 649, 871, 6171, 9663

## 結果

### 1. Pade近似テスト

| n | 最良近似 | 残差 |
|---|---------|------|
| 7 | [9/1] | 4.85e-07 |
| 27 | [8/2] | 5.79e-05 |
| 97 | [2/8] | 4.60e-06 |
| 171 | [9/1] | 9.94e-06 |
| 649 | [9/1] | 3.37e-04 |
| 871 | [2/8] | 2.28e-04 |
| 6171 | [8/2] | 1.26e-02 |
| 9663 | [6/4] | 6.66e-02 |

- 停止時間が短いn（n=7など）は残差が非常に小さい（軌道の大部分が周期{1,2}）
- 停止時間が長いn（n=6171, 9663）は残差が大きく、有理近似が困難
- **次数を上げても残差はゼロにならない** → 過渡部分の生成関数は非有理

### 2. 線形再帰テスト

| n | 結果 | 詳細 |
|---|------|------|
| 7 | 厳密再帰検出 | 次数12（1到達後の周期{1,2}を含む） |
| 27 | 近似再帰 | 次数46, 相対誤差 4.3e-04 |
| 97 | 検出されず | 次数20でも相対誤差 ~1.0 |
| 171 | 検出されず | 同上 |
| 649 | 検出されず | 同上 |
| 871 | 検出されず | 次数20で相対誤差 4.8e-02 |
| 6171 | 検出されず | 次数20で相対誤差 9.8e-01 |
| 9663 | 近似再帰 | 次数34, 相対誤差 9.4e-04 |

- n=7は停止時間11で軌道の大半が周期部分 → 容易に再帰が見つかる
- 過渡部分が長いnでは厳密な有限次線形再帰は存在しない
- **F(x)が有理関数でないことの直接的証拠**

|---|----------|-------|--------|
| 7 | 0.999 | 0.999 | 全て単位円内 |
| 27 | 0.923 | 0.923 | 全て単位円内 |
| 97 | 0.923 | 0.923 | 全て単位円内 |
| 171 | 0.923 | 0.923 | 全て単位円内 |
| 649 | 0.938 | 0.938 | 全て単位円内 |
| 871 | 0.895 | -0.365+0.817i | 全て単位円内 |
| 6171 | 0.948 | 0.948 | 全て単位円内 |

- **支配的極の|z|の平均: 0.929**, 標準偏差: 0.033
- 全ての極が単位円の内側（|z| < 1）にある
- これは軌道列が指数的に減衰する（= 1に収束する）ことと整合
- 多くのnで支配的極が実軸上正（|z| ≈ 0.92）に集中 → 普遍的減衰率の存在を示唆

### 4. Dirichlet級数

- 収束横軸 σ_c の平均: 0.40（軌道が最終的に有界であるため低い値）
- 停止時間の長いn（n=871）では σ_c ≈ 1.0 とやや高い

### 5. 対数軌道の解析

| n | 実測傾き | 傾き/log(3/4) |
|---|---------|--------------|
| 27 | -0.0140 | 0.049 |
| 97 | -0.0113 | 0.039 |
| 871 | -0.0474 | 0.165 |

- 実測傾きは理論値 log(3/4) ≈ -0.288 よりも絶対値が小さい
- これは軌道の前半で上昇期があり、全体平均が緩やかになるため
- 残差に明確な周期性は検出されず → カオス的振動

## 主要発見

### 発見1: 生成関数の構造的二重性
Syracuse軌道の生成関数は以下のように分解される:
```
F_n(x) = F_n^{trans}(x) + F_n^{periodic}(x)
```
- **F_n^{trans}(x)**: 過渡部分（1到達前）— **非有理的**（超越的）
- **F_n^{periodic}(x)**: 定常部分（1到達後の{1,2,1,2,...}）— **有理的** = (1+2x)/(1-x^2)

### 発見2: Z変換の極の普遍性
- 支配的極の|z|が0.88〜0.95の狭い範囲に集中
- 多くのnで実軸上 z ≈ 0.92 に支配的極が出現
- この値は軌道の平均減衰率 (3/4)^{1/2} ≈ 0.866 に近い

### 発見3: 線形再帰の不在と有理性の否定
- 過渡部分が長い軌道では有限次の線形再帰が存在しない
- これは偶奇遷移のカオス的性質に直接起因する
- 「全てのnで有限時間後に有理的定常状態に入る」ことがコラッツ予想と同値

### 発見4: Pade残差と停止時間の相関
- 停止時間が長いほどPade近似の残差が大きくなる
- 過渡部分の「複雑さ」は停止時間に比例して増大する

## コラッツ予想への含意

1. **生成関数のアプローチの限界**: F(x)の非有理性は「軌道が複雑」であることを示すが、「有限時間で1に到達する」ことの証明には直接つながらない。
2. **有望な方向**: Z変換の極が全て単位円内にあることは、軌道の減衰（1への収束）を示す強い数値的証拠。もし「全てのnについて全ての極が|z|<1」を証明できればコラッツ予想の証明に近づく。
3. **支配的極の普遍性**: |z| ≈ 0.92 という値が多くのnで共通することは、コラッツ力学系に普遍的な減衰メカニズムが存在することを示唆する。
