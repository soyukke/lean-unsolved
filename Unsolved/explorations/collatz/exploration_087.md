# 探索 087: 軌道の上昇/下降パターンの組合せ的数え上げ

## メタデータ
- **カテゴリ**: 組合せ論的
- **成果レベル**: 中発見
- **日付**: 2026-03-27

## アプローチ
Syracuse軌道をU(v2=1: 上昇)とD(v2≥2: 下降)の2文字列として符号化し、長さLの全2^L通りのU/D列のうち実際に軌道中に出現するものを数え上げた。禁止パターンの存在、U連続の上限、マルコフ連鎖としての遷移構造を体系的に分析。

## 発見

### 1. 禁止パターンは存在しない（L≤15で確認）
- L=1からL=14: 全てのU/D列が奇数n≤50万のSyracuse軌道中に出現（禁止パターン数=0）
- L=15: n≤50万では31パターンが未出現だが、n≤200万に拡張すると全32768パターンが実現
- **任意の有限U/D列はSyracuse軌道の部分列として実現可能**と推測される

### 2. U^k（v2=1のk連続）の完全な代数的構造
- U^k先頭の密度は厳密に1/2^kに一致（k≤14で精度4桁以上）
- **U^kを生成する剰余類**: n ≡ 2^k-1 (mod 2^(k+1))の正確に2個の剰余類
- U^k後の値の公式: T^k(n) = (3^k·n + c_k)/2^k, ここでc_k = Σ_{i=0}^{k-1} 3^{k-1-i}·2^i
- n = 2^k - 1 が最小のU^k先頭の奇数（例: 2^19-1=1048575で19連続U）
- **U連続に上限はない**: 任意のkで実現可能だが密度は指数的に減少

### 3. U/D列は2次マルコフ連鎖
- **1次遷移**: P(U|U)≈0.520, P(U|D)≈0.479（わずかに正の自己相関）
- **2次遷移（非対称性が顕著）**:
  - P(U|DU)≈0.571: D後のUは次もUになりやすい
  - P(U|UU)≈0.477: U連続後はUが出にくい（反発効果）
  - P(U|DD)≈0.454: D連続後はUが出にくい
  - P(U|UD)≈0.501: ほぼ均等
- カイ二乗検定: 統計量21773 >> 臨界値5.99 → **2次マルコフ性は極めて有意**

### 4. 4-gramの遷移確率パターン
- P(U|DDD)=0.461, P(U|DDU)=0.589: DDの後のUは次もUを引きやすい
- P(U|UUU)=0.457, P(U|UUD)=0.457: UUの後はDに傾く
- P(U|DUU)=0.495, P(U|DUD)=0.563: DU文脈では次のUが出やすい
- **全体として「均衡復元力」が働いている**: U連続はD方向へ、D連続はU方向へ引き戻される

### 5. 定常分布と理論値
- 1次マルコフの定常分布: π(U)≈0.500（U/Dの2値符号化ではほぼ均等）
- これはE[v2]=2.0と整合: v2=1の確率=1/2, v2≥2の確率=1/2
- **注意**: U率≈0.5はU/D符号での話。d/u比でlog2(3)を超えるのはDステップでのv2>2の寄与による

### 6. U連続とD連続の分布
- 最大U連続（n≤100000）: mode=5 (37.7%), 最大=16
- 最大D連続（n≤100000）: mode=5 (44.4%), 最大=12

## 仮説
- **禁止パターン不在の証明可能性**: mod 2^K のマルコフ連鎖がエルゴード的であることから、任意の有限U/D列が到達可能であることを証明できるはず
- **2次マルコフの「均衡復元力」**: U過剰→D方向への引き戻し、D過剰→U方向への引き戻しが、d/u > log2(3)を保証するメカニズムの一部
- **U^k密度の厳密な1/2^k**: v2(3n+1)の分布がn mod 2^(k+1)で完全に決まることの直接的帰結

## 行き止まり
- 禁止パターンによる漸化式の構築: 禁止パターンが存在しないため不成立
- U連続の上限の同定: 上限は存在しない（任意のkで実現可能）
- L=15でのサンプル不足: 最初31個の「禁止」はサンプル不足の偽陽性
