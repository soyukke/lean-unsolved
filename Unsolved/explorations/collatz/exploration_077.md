# 探索77-80: 5n+1 コラッツ変種

## 目的
3n+1 の代わりに 5n+1 としたとき、全ての奇数は1に到達するか？

## 主要発見

### 答え: NO
5n+1 変種では大半の数は1に到達しない。

### 構造的発見
1. **非自明サイクル2つ**:
   - サイクル1 (長さ10): 13→66→33→166→83→416→208→104→52→26→13
   - サイクル2 (長さ10): 17→86→43→216→108→54→27→136→68→34→17

2. **1に到達する奇数は極めて稀**: 奇数1..9999のうち78個のみ（1.56%）

3. **逆像ツリーの構造**:
   - レベル0: {1}
   - レベル1: {3, 51, 819, 13107} = {(2^(4k)-1)/5 : k=1,2,3,4}
   - 各レベルで逆像 m = (n·2^j - 1)/5 により拡大

4. **逆像条件**: m = (n·2^j - 1)/5 が奇数正整数 ⟺ n·2^j ≡ 1 (mod 5)
   - n ≡ 0 (mod 5): 解なし（5の倍数は逆像を持たない）
   - n ≡ 1: j ≡ 0 (mod 4)
   - n ≡ 2: j ≡ 3 (mod 4)
   - n ≡ 3: j ≡ 1 (mod 4)
   - n ≡ 4: j ≡ 2 (mod 4)

5. **フェルマーの小定理**: 2^4 ≡ 1 (mod 5) が全構造の根底

## 3n+1 との比較
| 性質 | 3n+1 | 5n+1 |
|------|------|------|
| 1到達率 | 100%（予想） | ~1.56% |
| サイクル | {1,2,4}のみ（予想） | 少なくとも2つの非自明サイクル |
| 発散 | なし（予想） | 大半が発散 |
| 増加率 | 3/2 ≈ 1.5 | 5/2 = 2.5 |

## 意義
5n+1 での「増加率 > 2」が本質的な違い。3n+1 では増加率 3/2 < 2 なので
「平均的に偶数ステップ（÷2）が奇数ステップ（×3/2）を上回る」が
5n+1 では逆転し、軌道は平均的に増大する。
