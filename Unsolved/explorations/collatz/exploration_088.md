# 探索088: Z_2上のSyracuse写像の周期点の完全分類

## メタデータ
- **カテゴリ**: 論理的/モデル理論
- **成果レベル**: 中発見
- **日付**: 2026-03-27
- **スクリプト**: `scripts/collatz_padic_periodic.py`

## 目的
2-adic整数 Z_2 上でSyracuse写像 T の周期点を全て分類する。
mod 2^k (k=1..27) での周期点列挙と、k→∞の極限でのリフト可能性を検証。

## アプローチ
1. mod 2^k (k=1..25) での全周期p(p=1..20)の周期点を完全列挙
2. 不動点・周期2点のリフト追跡（Hensel的分析）
3. 代数的分析：T^p(n)=n の方程式構造と分母の奇数性
4. T^p の2-adic微分による安定性分析
5. Baker定理の量的形式による正整数サイクルの上界

## 主要結果

### 1. mod 2^k での周期点の系統的列挙
- **不動点**: 全ての k=1..27 で n=1 が唯一の不動点（完全リフト可能）
- **周期2-5**: 全ての k=1..25 で存在せず
- **周期6**: k=12 でのみ偽のサイクル出現（871→1307→1961→1471→2207→3311）、k=13以降で消滅（リフト不可能）
- **周期7-20**: サイクルなし（k=12の周期6を除く）

### 2. 分母の奇数性定理（核心発見）
**定理**: 周期p方程式の分母 D = 2^s - 3^p は常に奇数（v2(D) = 0）

証明: v2(2^s - 3^p) = v2(-3^p) = 0（3^p は常に奇数、s ≥ 1 のとき 2^s ≡ 0 mod 2）

帰結:
- Z_2 上で分母は常に可逆 → 各 (v_1,...,v_p) に対して形式的な Z_2 上の解が1つ存在
- Z_2 上には各周期の形式的周期点が無限に存在する
- しかし正整数に属するものは計算的に存在しない

### 3. 不動点の代数的構造
T(n) = n ⟺ n = 1/(2^v - 3) で:
- v=1: n = -1（Z_2上の負の不動点）
- v=2: n = 1（唯一の正整数不動点）
- v≥3: 形式的 Z_2 の元 1/(2^v-3) は存在するが、v2(3n+1) の実際の値が v と不一致 → 真の不動点でない

### 4. 2-adic微分と安定性
n=1 での T の2-adic微分: v2(dT/dn)|_{n=1} = -2、つまり |dT/dn|_2 = 4 > 1
→ n=1 は **2-adic的に反発的不動点**（通常の力学系の不安定不動点に相当）

### 5. 正整数サイクルの上界排除
s = ⌈p·log₂3⌉ のとき |n_0| ≤ (3^p/2) / |2^s - 3^p|:
- p=2: 上界 0.64 < 1 → 正整数解なし（純代数的排除）
- p=4: 上界 0.86 < 1 → 正整数解なし
- p=7: 上界 0.57 < 1 → 正整数解なし
- 多くの p で最適な s を選ぶと上界 < 1

### 6. ord(3) mod 2^k の確認
ord(3) mod 2^k = 2^{k-2} (k≥3) を数値的に確認（k=1..24）

## Z_2上の周期点の完全分類表

| 周期p | Z_2上の形式的解 | 正整数解 | 排除方法 |
|-------|----------------|---------|---------|
| 1（不動点）| 無限個（各vに1つ）| n=1のみ | 代数的 |
| 2 | 存在 | なし | 上界 < 1 |
| 3 | 存在 | なし | 計算的排除 |
| 4 | 存在 | なし | 上界 < 1 |

## 新しい視点と発見

1. **Z_2 上の豊富な構造**: 分母 2^s - 3^p が常に奇数であることから、Z_2 上には各周期の周期点が無限に存在する。コラッツ予想の難しさは「Z_2 上の解のうち、どれが正整数に属するか」という篩い分けの問題に帰着される。

2. **反発的不動点**: n=1 が 2-adic的に反発的（|dT/dn|_2 = 4）であることは、Z_2 上で 1 の近傍の点が T の反復で 1 から離れることを意味する。これは通常の距離での収束（コラッツ予想）と対照的。

3. **mod 2^k でのリフト障害**: k=12 の周期6サイクルが k=13 にリフトできない現象は、Hensel条件の失敗を示す具体例。T^6 の微分が k=12 で 0 mod 2 となり、リフトが分岐・消滅する。

## 形式化への道筋
- `v2(2^s - 3^p) = 0`（分母の奇数性）は `Nat.odd_pow` 等で形式化可能
- 不動点方程式の一意性は CollatzCycle.lean で既に形式化済み
- 周期2の代数的排除（上界 < 1）は算術不等式として形式化可能
