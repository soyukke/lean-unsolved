# 探索052: コラッツ写像の反復関数系(IFS)解析

## 日付
2026-03-18

## 目的
コラッツ写像を IFS (Iterated Function System) {f_0(x) = x/2, f_1(x) = (3x+1)/2} として捉え、アトラクタ構造、Hausdorff次元、不変測度、リアプノフ指数、逆IFSの被覆性を解析する。

## 手法
- **ランダム反復法** (Chaos game): 確率 p で f_0、1-p で f_1 を選択し50万回反復
- **ボックスカウンティング法**: アトラクタの fractal 次元推定
- **リアプノフ指数**: λ(p) = p ln(1/2) + (1-p) ln(3/2) の理論値と実軌道からの推定
- **逆IFS木**: g_0(x)=2x, g_1(x)=(2x-1)/3 のBFS展開（深さ30）
- **Hutchinson作用素**: 区間への反復作用の観察
- **符号列解析**: 実コラッツ軌道の偶奇パターンとリアプノフ指数の関係

## 結果

### 1. リアプノフ指数の理論値と臨界確率

| p | λ(p) | 状態 |
|---|-------|------|
| 0.30 | +0.0759 | 発散 |
| p_c = 0.3691 | 0.0000 | 臨界 |
| 0.40 | -0.0340 | 収束 |
| 0.50 | -0.1438 | 収束 |
| p_bal = 0.6131 | -0.2681 | 収束 |
| 0.70 | -0.3636 | 収束 |

- **臨界確率**: p_c = ln(3/2)/ln(3) ≈ 0.3691（λ=0の境界）
- **均衡確率**: p_bal = log₂3/(1+log₂3) ≈ 0.6131（自然数の偶奇比に基づく）

### 2. 実軌道からのリアプノフ指数

| 初期値 | λ推定値 |
|--------|---------|
| 7 | -0.1463 |
| 27 | -0.1473 |
| 97 | -0.1473 |
| 871 | -0.1472 |

- 実軌道のリアプノフ指数は初期値によらず **λ ≈ -0.147** に収束
- これは p=0.5 の理論値 λ(0.5) = -0.1438 に極めて近い

### 3. IFS アトラクタの構造

| 確率 p | 平均 | 標準偏差 | BC次元 | 測度エントロピー |
|--------|------|---------|--------|----------------|
| 0.50 (等確率) | 18.04 | 275.10 | 0.633 | 0.059 |
| 0.6131 (均衡) | 1.70 | 5.39 | 0.685 | 0.364 |
| 0.3691 (臨界) | (発散) | (発散) | 0.539 | 0.009 |

- **等確率 p=0.5**: アトラクタは [0, ~55000] に広がるが、大部分は 0 付近に集中（中央値2.24）
- **均衡確率**: アトラクタは [0, ~870] に収まり、測度は 0 付近に集中
- **臨界確率**: 理論的にはλ=0だが、Python整数の無限精度により発散的振る舞いを示す

- 2^18 = 262144 に対して 204607 点 → 約22%の点が重複

### 6. 逆IFS木の成長と被覆

- 逆IFS木の成長率は **4/3 ≈ 1.333** に収束（探索049と整合）
- 深さ30で累積15,066ノード

| 範囲 | 被覆率 |
|------|--------|
| [1, 100] | 84.0% |
| [1, 500] | 58.8% |
| [1, 1000] | 50.4% |
| [1, 5000] | 29.1% |
| [1, 10000] | 21.9% |

- 深さ30では [1,1000] に496個の未到達数が残る
- 逆IFS木が全正整数を生成 ⟺ コラッツ予想（より深い展開が必要）

### 7. Hutchinson 作用素

- 初期区間 [0, 10] に対してHutchinson作用素を20回反復
- **常に単一区間**のまま（分裂しない）
- 区間長は各反復で約1.5倍に成長: 因子 = (1/2 + 3/2) / 2 の効果
- f_1 の像が f_0 の像と重なるため、CantoR集合的構造は現れない

### 8. 符号列と軌道の統計

1..10000 の全軌道の統計:
- **奇数ステップ比率（1の比率）**: 平均 0.4547、中央値 0.4717
- **軌道リアプノフ指数**: 平均 -0.1936、**全て負**
- **経験的偶数ステップ割合**: p_emp ≈ 0.545

個別例:
- n=27: 1の比率=0.586 (高い → 軌道が長くなる)
- n=127: 1の比率=0.484 (適度 → 中程度の軌道長)

## 考察

### IFS視点からのコラッツ予想の本質

1. **リアプノフ指数による収束判定**: コラッツ写像のIFS表現で、臨界確率 p_c ≈ 0.3691 を超えれば軌道は平均的に縮小する。実データでは p_emp ≈ 0.545 であり、p_c を大きく上回る。これはコラッツ予想が「ほぼ確実に」成り立つことの確率的根拠。

2. **BC次元の相転移**: p < p_c で発散、p > p_c で有界アトラクタ。臨界点付近で相転移が起きる。コラッツ写像はこの臨界点から十分離れた「安全側」にある。

3. **Hutchinson作用素の非分裂**: f_0, f_1 の像が常に重なるため、古典的なCantor集合型アトラクタは生成されない。代わりに [0, ∞) の部分区間上に集中する測度が不変測度となる。

4. **逆IFSとの双対性**: 順方向IFSの収束（λ < 0）と逆IFS木の全整数生成は表裏一体。順方向で「全ての軌道が1に到達」⟺ 逆方向で「1から全整数に到達」。

### 新しい知見

- **実軌道リアプノフ指数 λ ≈ -0.147** は理論値 λ(0.5) = -0.144 にほぼ一致。これは自然数における偶奇の分布が等確率に近いことの反映。
- **全1..10000の軌道でリアプノフ指数が負** という事実は、個々の軌道レベルでの収束の強い証拠。
- BC次元 ≈ 0.71（決定論的）は、IFSとしてのコラッツ写像のフラクタル的複雑さを定量化する新しい指標。
