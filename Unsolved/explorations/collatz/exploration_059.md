# 探索59: v2(3n+1) の幾何分布の厳密な証明

**日付**: 2026-03-18
**スクリプト**: `scripts/collatz_v2_distribution_proof.py`
**ステータス**: ✅完了

## 目的

v2(3n+1) が奇数 n に対して幾何分布 Geom(1/2) に従うことを厳密に証明し、E[v₂] = 2.0 を確立する。

## 主要結果

### 1. 厳密な定理

**定理**: mod 2^k の奇数 r (2^{k-1} 個) のうち v2(3r+1) = j のものは正確に 2^{k-1-j} 個 (j=1,...,k-1)。

**証明**:
1. r が奇数 ⟹ 3r+1 は偶数 ⟹ v2(3r+1) ≥ 1 は自明
2. v2(3r+1) ≥ j+1 ⟺ 2^{j+1} | (3r+1) ⟺ r ≡ 3^{-1}·(-1) (mod 2^{j+1})
3. 3 は mod 2^{j+1} で可逆、解 c は奇数、2^{j+1} は偶数
4. c + m·2^{j+1} は全て奇数 → #{v2 ≥ j+1} = 2^{k-j-1}
5. #{v2 = j} = 2^{k-j-1} - 2^{k-j-2} = 2^{k-1-j} (j < k-1)

**系**: P(v2 = j) = 2^{k-1-j} / 2^{k-1} = 1/2^j

### 2. E[v₂] の厳密値

E[v₂] = Σ_{j=1}^{k-1} j/2^j + k/2^{k-1} → 2.0 (k→∞)

### 3. v2 ≥ k の唯一の残基

v2(3r+1) ≥ k となる奇数 r は mod 2^k でちょうど1個:
- k 偶数: r₀ = (2^k - 1)/3
- k 奇数: r₀ = (2^{k+1} - 1)/3 mod 2^k
- 列: 1, 5, 5, 21, 21, 85, 85, 341, 341, ...（= 0101...01 の2進パターン）

### 4. 帰納法の核心

k → k+1 の帰納ステップ:
- j < k のとき: v2(3r+1) の値は mod 2^k → mod 2^{k+1} で保存
- 理由: v2(3r + 3·2^k) = v2(3r) + v2(1 + 2^k/(3r)) ... 実際は v2(a+b) の最小性から
- 数値検証: k=2..11 で完全に確認

### 5. Lean形式化の方針

- k=2,...,6 の有限ケースは `decide` で即座に証明可能
- 一般 k: `v2_count_eq : ∀ k j, j < k → #{r ∈ Fin(2^k) | r odd ∧ v2(3*r+1) = j} = 2^{k-1-j}`
- 帰納ステップには `v2_add_of_ne` 型の補題が必要

## コラッツ予想への含意

- E[v₂] = 2.0 > log₂(3) ≈ 1.585 は「ほとんど全ての軌道が下降する」の中核的根拠
- マージン E[v₂] - log₂(3) ≈ 0.415 は変種比較（探索48）の結果と整合
- 「平均→全て」のギャップが未解決問題の本質
