# 探索 080: Syracuse写像の mod 2^k 上の層別全単射構造

## 概要
Syracuse写像 T(n) = (3n+1)/2^{v2(3n+1)} が mod 2^k の奇数上で誘導する写像の構造を k=3..13 で調査。T は mod 2^k 上で well-defined な「置換」にはならないが、v2(3n+1) で層別すると各層上で完全な全単射が成立するという「階段的全単射構造」を発見・証明した。

## 手法
1. T(n) mod 2^k の well-definedness を検証
2. v2(3n+1) による奇数残基の層別分類
3. 各層上での T の写像構造の分析
4. 遷移確率行列の構造と定常分布の計算
5. 代数的証明の構成

## 発見

### 発見1: T は mod 2^k 上で well-defined でない
Syracuse写像 T(n) は奇数 mod 2^k 上で well-defined な写像にならない。原因は v2(3n+1) が n mod 2^k だけでは一意に定まらないこと。全ての k=3..12 で確認。

### 発見2: ★ v2(3n+1) の完全な分布公式
奇数 mod 2^k のうち v2(3n+1) = j となるものの数:
- j = 1, ..., k-2: **2^{k-1-j} 個**
- j = k-1: **1 個**
- j >= k: **1 個**

合計: 2^{k-2} + 2^{k-3} + ... + 2 + 1 + 1 + 1 = 2^{k-1} ✓

### 発見3: ★★ 階段的全単射定理
**定理**: v2(3n+1) = j (j < k) の層上で、T(n) mod 2^{k-j} は n mod 2^k のみで確定的に決まり、奇数 mod 2^{k-j} への全単射を構成する。

k=3..13 の全 k、全 j で検証済み。

**証明**:
- n ≡ r (mod 2^k), v2(3r+1) = j とする
- 3n+1 = 2^j·q + 3m·2^k = 2^j(q + 3m·2^{k-j}), q は奇数
- k-j > 0 のとき 3m·2^{k-j} は偶数 → v2(3n+1) = j が確定
- T(n) mod 2^{k-j} = (3r+1)/2^j mod 2^{k-j} (r のみに依存)
- 単射性: T(r) ≡ T(r') (mod 2^{k-j}) ⟺ 3(r-r') ≡ 0 (mod 2^k) ⟺ r ≡ r'
- |Layer(j)| = 2^{k-1-j} = |奇数 mod 2^{k-j}| → 全単射 ✓

### 発見4: 平均情報損失 = 2 bits
T の1ステップでの平均情報損失（2-adic valuation の期待値）:
- E[v2] = Σ j·2^{k-1-j}/2^{k-1} = 2 - (k+2)/2^k → 2 (k→∞)
- k=8: E[v2] ≈ 1.992, k=15: E[v2] ≈ 1.9999

### 発見5: ★ 遷移確率行列は一様分布に収束
T を確率的写像と見た遷移行列（各残基→compatible targets への一様遷移）は、定常分布が完全に一様分布と一致する。k=3,4,5 で確認。

### 発見6: T_1 のアフィン公式
v2=1 ステップのみの反復 T_1(n) = (3n+1)/2 について:
- T_1^j(n) = (3^j(n+1) - 2^j) / 2^j = (3/2)^j(n+1) - 1
- 閉じた式: c_j = 3^j - 2^j（漸化式 c_{j+1} = 3c_j + 2^j の解）

### 発見7: ord(3) mod 2^k = 2^{k-2}
3 の mod 2^k における乗法的位数は 2^{k-2}（k >= 3）。k=3..9 で検証。これは v2=1 層の置換構造の周期性に直結する。

### 発見8: v2=1 層の明示的写像パターン
n ≡ 3 (mod 4) のとき T(n) = (3n+1)/2。mod 2^{k-1} での像は:
- T(4m+3) = 6m+5
- 連続的にシフトする構造: T(3)≡5, T(7)≡11, T(11)≡17, ... (mod 2^{k-1})
- 各2倍ステップごとにラップアラウンドが発生

## 仮説
1. 階段的全単射構造は2-adic整数環 Z_2 上の T の連続性の有限版現れ
2. 一様定常分布は Collatz軌道の「エルゴード性」の代数的裏付け
3. 平均情報損失2 bits = 平均収縮率 3/4 = (3/2)/2 と整合（log_2(3/4) ≈ -0.415）

## 行き止まり
- T を mod 2^k 上の「置換」として分析する当初の方針は不成立（well-defined でないため）
- v2=1 層を奇数 mod 2^{k-1} 上の置換として巡回分解しようとしたが、domain と codomain の mod が異なるため直接的な置換にならない
