# 探索14: sumset の等差数列公式

## 目標
等差数列 {1,...,n} の sumset カード = 2n-1 のパターンを n=6 まで拡張検証する。

## 背景
- {1,2,3}: |A+A| = 5 = 2*3-1 (探索3)
- {1,...,4}: |A+A| = 7 = 2*4-1 (探索10)
- {1,...,5}: |A+A| = 9 = 2*5-1 (探索12)

## 結果
- {1,...,6}: |A+A| = 11 = 2*6-1 を `decide` で検証成功
- 6元集合（36ペア）でも `decide` は問題なく通る

## Lean定理
- `sumset_123456_card`: `(sumsetFinset {1, 2, 3, 4, 5, 6}).card = 11`

## 考察
等差数列 {1,...,n} の sumset は {2, 3, ..., 2n} であり、カードは常に 2n-1。
これは sumset が最小となる集合の典型例であり、Freiman-Ruzsa理論における
倍加定数 |A+A|/|A| ≈ 2 の最適例。
