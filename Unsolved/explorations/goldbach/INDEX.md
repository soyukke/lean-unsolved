## 主な発見

### 探索1: Mathlib調査
- ゴールドバッハ予想の既存形式化はMathlibに存在しない
- `norm_num` が小さい素数の判定に使える
- `Nat.Prime.odd_of_ne_two` など、奇素数の性質が利用可能

### 探索2: 数値実験の結果
- 4 ≤ n ≤ 10000 の全4999個の偶数で予想が成立
- g(n)=1（唯一の素数ペア）は n=4,6,8,12 の4個のみ
- n ≡ 0 (mod 6) の偶数は g(n) が特に大きい（平均127 vs 62-65）
- Hardy-Littlewood 近似 g(n) ~ C₂·n/(ln n)² と概ね一致（比率0.8-1.0）

### 探索3: Lean形式化
- `IsGoldbach` 述語を定義
- 2つの奇素数の和が偶数であることを証明 (`even_of_odd_prime_add`)
- `isGoldbach_of_prime_add`: 素数ペアからゴールドバッハ性を導出

### 探索4: mod 6/30 構造
- **mod 6 が g(n) を強く支配**: n≡0 (mod 6) は平均g(n)=127.3、n≡2,4 は 62-66
- 比率 g(n≡0)/g(n≡2) ≈ 2.04 で全区間でほぼ一定
- **(p%6, q%6) パターンが決定的**:
  - n≡0 (mod 6): (5,1)と(1,5) がほぼ50%ずつ → 2パターン使える
  - n≡2 (mod 6): (1,1) が 99.4% → 1パターンのみ
  - n≡4 (mod 6): (5,5) が 99.4% → 1パターンのみ
- mod 30 構造: n≡0 (mod 30) が最大の平均g(n)=158.9（5の倍数の追加効果）
- **結論**: g(n)の約2倍差は (p%6, q%6) の利用可能パターン数の差で説明可能

### 探索5: Hardy-Littlewood 予想式
- g(n)/HL(n) の平均比率は 0.644（p≤qのペアのみ数えるため HL 式の約半分）
- **Singular series S(n) の効果が明確に確認**:
  - S(n)=1 (2べき): g(n)が最小
  - S(n)が大きい偶数 (多くの奇素因子): g(n)が顕著に大きい
  - S(n)=3.56 の最大値は n=2310=2*3*5*7*11 で達成
- g(n)/(C₂*n/(ln n)²) と S(n) の相関は良好（比率が全S範囲で約0.64で安定）
- 収束性: n が大きくなるにつれ標準偏差が減少（0.093→0.024）

### 探索6: 弱いゴールドバッハ予想
- `IsWeakGoldbach n`: 3つの素数の和で表せる述語を定義
- `WeakGoldbachConjecture`: 全奇数 n > 5 が3つの素数の和で表せる
- 5つの検証例: 7=2+2+3, 9=3+3+3, 11=3+3+5, 15=3+5+7, 21=3+5+13
- **`goldbach_implies_weak_goldbach`**: ゴールドバッハ予想 → 弱い版の含意を証明
  - n > 5 かつ奇数 → n-3 > 2 かつ偶数 → GC適用 → n = 3 + p + q

### 探索7: mod 6 構造定理
- `prime_gt3_mod6`: 3より大きい素数は mod 6 で 1 か 5（Goldbach.lean 内で自己完結的に証明）
- `goldbach_mod6_implies_both_1`: n≡2 (mod 6) かつ p+q=n かつ p,q>3 素数 → p≡q≡1 (mod 6)
- 4通りの場合分けで (1,1) のみが n%6=2 と整合することを示す
- n≡2 (mod 6) が「最も難しい」ケースである構造的説明を提供

### 探索8: 追加検証例
- 200 = 3 + 197, 500 = 13 + 487, 1000 = 3 + 997
- 497 は合成数 (7×71) のため 500 の分解には 13 + 487 を使用
- 累計検証例: 4, 6, 8, 10, 12, 20, 100, 200, 500, 1000 の10個

### 探索9: n ≡ 0 (mod 6) の構造定理
- `goldbach_mod6_zero`: n≡0 (mod 6) かつ p+q=n (p,q>3素数) → (p%6=1∧q%6=5) ∨ (p%6=5∧q%6=1)
- 1+5=6≡0, 5+1=6≡0 のみ成立。1+1=2, 5+5=10≡4 は不適

### 探索10: n ≡ 4 (mod 6) の構造定理
- `goldbach_mod6_four`: n≡4 (mod 6) かつ p+q=n (p,q>3素数) → p%6=5 ∧ q%6=5
- 5+5=10≡4 のみ成立
- **mod 6 構造定理の完成**: n%6=0→(1,5)or(5,1), n%6=2→(1,1), n%6=4→(5,5)

### 探索11: IsGoldbach の偶数性
- `isGoldbach_odd_has_two`: IsGoldbach n かつ n が奇数 → ∃ q 素数, n = 2 + q
  - 2つの奇素数の和は偶数なので、奇数の IsGoldbach では必ず p=2 または q=2
  - even_of_odd_prime_add を活用、Nat.Even.not_odd で矛盾を導出
- `isGoldbach_even_ge_four`: IsGoldbach n かつ偶数 → n ≥ 4
  - 各素数は ≥ 2 なので和は ≥ 4

### 探索12: 大きい検証例 + isGoldbach_double_prime
- 2000 = 3 + 1997, 5000 = 7 + 4993, 10000 = 59 + 9941
- norm_num で4桁素数も問題なく判定（各数秒以内）
- `isGoldbach_double_prime`: p が素数なら 2p は IsGoldbach
  - 素数の無限性と合わせて IsGoldbach な偶数が無限に存在
- 累計検証例: 4, 6, 8, 10, 12, 20, 100, 200, 500, 1000, 2000, 5000, 10000 の13個

### 探索13: 素数ペアの一意性 + isGoldbach_sum_comm
- 10 = 5+5 の別分解（既存の 3+7 と合わせて2通り）
- `isGoldbach_sum_comm`: 素数 p, q に対して IsGoldbach (q + p) が成立
- ゴールドバッハ分解の非一意性の具体例を提供
