# 探索13: 距離の加法性

## 日付
2026-03-18

## 目的
原点からの距離の二乗 distSq(0, p) が二次形式であることを示す展開公式を証明する。

## 手法
distSq (0,0) (x1+x2, y1+y2) を展開し、distSq (0,0) (x1,y1) + distSq (0,0) (x2,y2) + 交差項 の形で表す。
これは内積 (ユークリッド内積) の存在を反映した恒等式。

## 結果

### Lean 定理
```lean
/-- distSq は (0,0) に対して二次形式 -/
theorem distSq_origin_add (x1 y1 x2 y2 : ℤ) :
    distSq (0, 0) (x1 + x2, y1 + y2) =
    distSq (0, 0) (x1, y1) + distSq (0, 0) (x2, y2) + 2 * (x1 * x2 + y1 * y2) := by
  unfold distSq; ring
```

## 考察
- この等式は ||u+v||² = ||u||² + ||v||² + 2⟨u,v⟩ の整数版
- 交差項 2*(x1*x2 + y1*y2) はユークリッド内積の2倍
- 特に直交する場合 (x1*x2 + y1*y2 = 0) にはピタゴラスの定理 ||u+v||² = ||u||² + ||v||² が成立
- この性質は格子点上の距離分布を理解する上で基本的
- ブラーマグプタ-フィボナッチ恒等式 (sum_sq_mul_sum_sq) と合わせて、距離の代数構造が充実
