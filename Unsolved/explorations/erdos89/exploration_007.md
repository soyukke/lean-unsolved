# 探索7: 共線点集合の異なる距離

## 目的
同一直線上の等間隔 n 点が持つ異なる距離を具体的に検証する。

## 主要結果
- 3点 (0,0), (1,0), (2,0) の距離²は {1, 4} の2つ
- 4点 (0,0), (1,0), (2,0), (3,0) の距離²は {1, 4, 9} の3つ
- 一般に n 点の等間隔共線配置は n-1 個の異なる距離を持つ

## 証明方法
`decide` タクティクで具体的な距離²の値を計算・検証。

## 所見
共線配置は異なる距離が n-1 個で、二次元格子配置より多い。
これはエルデシュ予想の下界 Ω(n/√log n) を大きく超えており、
共線配置が最小異なる距離の意味で非効率であることを示す。
n-1 = Θ(n) であり、二次元格子の Θ(n/√log n) と対照的。
