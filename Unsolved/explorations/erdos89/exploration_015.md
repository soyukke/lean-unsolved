# 探索15: ピタゴラス三つ組の距離

## 目的
ピタゴラス三つ組 (a,b,c) で a²+b²=c² となるものについて、
distSq を用いて距離²の計算を検証する。

## 内容

### 検証した三つ組
1. **(3, 4, 5)**: distSq (3,0) (0,4) = 9 + 16 = 25 = 5²
2. **(5, 12, 13)**: distSq (5,0) (0,12) = 25 + 144 = 169 = 13²
3. **(8, 15, 17)**: distSq (8,0) (0,15) = 64 + 225 = 289 = 17²

### 意義
- 整数格子上の異なる距離問題では、2つの平方和で表せる数が鍵
- ピタゴラス三つ組は distSq が完全平方数になるケース
- これらの距離²は同時に m² + n² の形にも書ける（例: 25 = 3²+4² = 5²+0²）
- 距離の多重表現が格子上の異なる距離数を減らす要因

### 証明
- 全て `decide` で自動検証（整数リテラルの算術）

## 結果
- 3つの example が sorry なしでビルド成功
- 探索10で検証した (3,4,5), (5,12,13) に加え、新たに (8,15,17) を追加
