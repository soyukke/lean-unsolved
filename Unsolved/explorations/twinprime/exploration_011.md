# 探索11: 大きい双子素数の検証

## 日付
2026-03-18

## 目的
3桁の双子素数ペアを norm_num で検証し、検証範囲を拡大する。

## 手法
100以上の双子素数ペアを選定:
- (101, 103): 101 は素数、103 は素数
- (107, 109): 107 は素数、109 は素数
- (137, 139): 137 は素数、139 は素数
- (149, 151): 149 は素数、151 は素数

全て `by constructor <;> norm_num` で検証。

## 結果

### Lean 検証例
```lean
-- (101, 103) は双子素数
example : Nat.Prime 101 ∧ Nat.Prime 103 := by constructor <;> norm_num

-- (107, 109) は双子素数
example : Nat.Prime 107 ∧ Nat.Prime 109 := by constructor <;> norm_num

-- (137, 139) は双子素数
example : Nat.Prime 137 ∧ Nat.Prime 139 := by constructor <;> norm_num

-- (149, 151) は双子素数
example : Nat.Prime 149 ∧ Nat.Prime 151 := by constructor <;> norm_num
```

## 考察
- norm_num は3桁の素数判定に問題なく対応
- 累計検証済み双子素数: (3,5), (5,7), (11,13), (17,19), (29,31), (41,43), (59,61), (71,73), (101,103), (107,109), (137,139), (149,151) の12組
- 100以下に8組、100-150に4組。密度の減少が見て取れる
- 次の双子素数は (179,181), (191,193), (197,199) など
